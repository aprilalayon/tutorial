Learn how to code it below.

Step 1

Download the Progress Bar master file.

Step 2

Include the jQuery library to the scripts section of your HTML.

    https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js

Step 3

Find and include the dist/progressbar.min.js from the master file to your scripts section.

Step 4

Follow the HTML markup below to create your circle containers.

    
    <div class="circle-container">
        <div id="htmlContainer"><h3 class="html">HTML</h3></div>
        <div id="cssContainer"><h3 class="css">CSS</h3></div>
         <div id="javaContainer"><h3 class="java">Javascript</h3></div>
    </div>

Step 5

Using CSS, give your circular progress bars a width and a height. You can also position your circular progress bars and labels however you like.

    
    #htmlContainer,
    #cssContainer,
    #javaContainer {
        width: 200px;
        height: 200px;
        position: relative;
    }

    .html,
    .css,
    .java {
        position: absolute;
        top: 50%;
        left: 50%;
        transform: translate(-50%, -50%);
    }

Step 6

Now we just have to write a few scripts in your scripts section or file to make your circular progress bars animate. You can also customize the appearance, duration, and the length of your progress bar.

    
    var bar = new ProgressBar.Circle(htmlContainer, {
       strokeWidth: 10,
       easing: 'easeInOut',
       duration: 1400,
       color: '#FFEA82',
       trailColor: '#eee',
       trailWidth: 10,
       svgStyle: null
     });

     bar.animate(0.65);
          
    var bar = new ProgressBar.Circle(cssContainer, {
      strokeWidth: 10,
      easing: 'easeInOut',
      duration: 1400,
      color: '#FFEA82',
      trailColor: '#eee',
      trailWidth: 10,
      svgStyle: null
    });

    bar.animate(0.65);

    var bar = new ProgressBar.Circle(javaContainer, {
      strokeWidth: 10,
      easing: 'easeInOut',
      duration: 1400,
      color: '#FFEA82',
      trailColor: '#fff',
      trailWidth: 10,
      svgStyle: null
    });

    bar.animate(.30);
