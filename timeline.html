Step 1

Download the Timelify master file.

Step 2

Include the jQuery library to the scripts section of your HTML.

    https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js

Step 3

Find and include the jquery.timelify.min.js from the master file to your scripts section.

Step 4

Find and include the timelify.css and animate.css from the master file and link it to your head section.

Step 5

Follow the HTML markup below to create your timeline.

    
   <div class="container">
           <div class="timeline">
               <h2>2017</h2>
               <ul class="timeline-items">
                   <li class="is-hidden timeline-item"> 
                       <h3>Project Title</h3>
                       <hr>
                       <p>Description of the project goes here.</p>
                       <hr>
                       <p>January 2016</p>
                   </li>
               </ul>
               <h2>2016</h2>
               <ul class="timeline-items">
                   <li class="is-hidden timeline-item inverted"> 
                       <h3>Project Title</h3>
                       <hr>
                       <p>Description of the project goes here</p>

                       <hr>
                       <p>March 2016</p>
                   </li>
                   <li class="is-hidden timeline-item"> 
                       <h3>Project Title</h3>
                       <hr>
                       <p>Project description goes here.
                       </p>

                       <hr>
                       <p>April 2016</p>
                   </li>
               </ul>
                       
           </div>
   </div>

Step 6

Position the container of your timeline and style your text using CSS. 

    
     .container {
       max-width: 960px;
       margin: 0 auto;
       padding: 40px;

     }

     .timeline h3 {
         text-align: center;
         font-size: 20px;
     }

     .timeline p {
         font-size: 15px;
         color: white;
         line-height: 1.4;
     }

Step 7

In your scripts section or file, write the following code. You can customize it to your liking.

    
      $('.timeline').timelify({
          animLeft: "fadeInLeft",
          animRight: "fadeInRight",
          animSpeed: 600,
          offset: 150
      });
