Type out an introduction about yourself for your audience to read. Just keep in mind to keep it short and sweet!

Learn how to code it below.

Step 1

Download the Typed master file.

Step 2

Include the jQuery library to the scripts section of your HTML.

    https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js

Step 3

Find and include the dist/typed.min.js from the master file to your scripts section.

Step 4

All you really need is this HTML markup to provide a space for your type box.

    
    <span class="text"></span>

Step 5

You are able to style your text by playing around with the placement and properties like font-family, color, text-shadow etc.

    
    .text {
        font-family: 'Ubuntu', sans-serif;
        font-size: 40px;
        font-weight: bold;
        text-align: center;
        color: #ffffe0;
        display: inline;
        text-shadow: 2px 1px 0px rgba(150, 150, 150, 1);
        line-height: 0.8;
    }

Step 6

Type in the following script to make your type effect work. If you changed the class name in your HTML markup, make sure to match it in your script. 

    
    $(function repeat(){
        $(".text").typed({
            strings: ["Type anything you like here."],
            typeSpeed: 100
        
        });
         
    });

Step 7 - Optional

If you would like customize the blinking cursor. Type the following in your CSS code. Again, you can customize the size of your cursor as well as the color to your liking.

    
    .typed-cursor{
        font-size: 60px;
        color: lightyellow;
        opacity: 1;
        -webkit-animation: blink 0.7s infinite;
        -moz-animation: blink 0.7s infinite;
        animation: blink 0.7s infinite;
    }
    
    @keyframes blink{
        0% { opacity:1; }
        50% { opacity:0; }
        100% { opacity:1; }
    }
    @-webkit-keyframes blink{
        0% { opacity:1; }
        50% { opacity:0; }
        100% { opacity:1; }
    }
    @-moz-keyframes blink{
        0% { opacity:1; }
        50% { opacity:0; }
        100% { opacity:1; }
    }
